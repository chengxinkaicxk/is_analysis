# 接口:queryLabTitle [返回](../用例/查看实验课题用例.md)

* 功能:返回某一课程的所有实验课题
* API请求地址:接口基本地址/cxk/api/queryLabTitle
* 请求方式:GET
* 请求参数说明:无
* 返回实例:
<pre>
{
      "code": 200,
      "msg": null,
      "data": [
          {
            "test_id":"1",
            "title":"实验1",
            "content":"实验要求......",
            "end_time":"截止日期"
            },{
              ...其他实验
          }
      ]
  }
</pre>

- 返回参数说明：

  |参数名称|说明|
  |:---------:|:-------------------------|
  |code|返回的状态码，200为正常|
  |msg|返回的信息|
  |test_id|返回的实验编号|
  |title|返回的实验课题|
  |content|返回的实验的实验要求|
  |end_time|返回的实验的截止日期|
